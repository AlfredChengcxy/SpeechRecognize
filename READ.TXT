先建立数据库
create database web;
use web;
create table data (id int(4) auto_increment not null primary key, 
logname varchar(50),
em varchar(50));

,或更改数据库的连接代码,
你也可以在DOS的命令行中输入mysql -uroot -p密码 < c:\...\iNENS.sql 
(小心此DOS动作将删除库 web 如果你有的话)
然后将目录class下的所有文件拷贝至:\Tomcat40\webapps\examples\WEB-INF\classes下,
将目录jsp下的所有文件拷贝至:\Tomcat40\webapps\examples\jsp下,
在浏览其中输入http://localhost:8080/examples/jsp/index.htm即可.
